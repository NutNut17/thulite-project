HTML COLORS

Color code are written as style="color:colourname/colourcode"

RGB code(Red Green Blue) "style=color:rgb(x,y,z)". Where x,y,z ranged in 0-255 and x is red light source, y is green light source, z is blue light source. or example:rgb(60, 179, 133)rgb(5, 5, 5)

RGBA code(RGB with Alpha channel)"style=color:rgba(x,y,z,a)". The alpha,a determine the opacity of the color.
The colour is fully transparent when a=0 andfully opaque when a=1. For example:rgb(255, 0, 0,1) rgb(255, 0, 0,0.5)

HEX code(Hexadecimal Colors)"style=color:#rrggbb, r for red, g for green, b for blue. The value of rr,gg,bb ranged from 00-ff. For example:
#ff0000#6a5acd #ffa500#ffffff

HSL code(Hue Saturation Lightness)"style=color:hsl(x,y,z)". Where y,z valued from 0-100%. X is the Hue(0-360),0 is red,120 is green,
240 is blue from color wheel. Y is saturation of the color, 0% is grey, 100% is the full color. Z is lightness of the color.
0% is black, 100% is white. For example:hsl(248, 53%, 58%) hsl(60, 0%, 58%)hsl(220, 50%, 20%)

HSLA(HSL with Alpha channel)"style=color:hsla(x,y,z,a)".

HTML CSS

Cascading Style Sheets (CSS) is used to format the layout of a webpage.CSS can be added to HTML documents in 3 ways:
Inline - by using the style attribute inside HTML elements
Internal - by using a "style" element in the section
External - by using a "link" element to link to an external CSS file

HTML LINKS

Use "a" element to make a link.For example:Click Here to W3School.
The title attribute shows a text when a mouse over it and the href attributes link to a website.

-To use a full URL to link to a web page, write as https://www.w3schools.com/html/default.asp in href
-To link to a page located in the html folder on the current web site, write as /html/default.asp in href
-To link to a page located in the same folder as the current page, write as default.asp in href

The target attribute specifies where to open the linked document.The target attribute can have one of the following values:
"_self" - Default. Opens the document in the same window/tab as it was clicked
"_blank" - Opens the document in a new window or tab
"_parent" - Opens the document in the parent frame
"_top" - Opens the document in the full body of the window

An image also can be used to link. To do this, just add "img" element inside "a" element

Insert mailto:(email) inside the href attribute to create a link that opens the user's email program (to let them send a new email)
For example:Send email

For the color of links and it's decoration, look intoHTMLstyles.ccs located together with this document

Link also can be displayed as button by adding such style elements in the head elements as:
a:link, a:visited {background-color: #f44336;color: white;padding: 15px 25px;
text-align: center;text-decoration: none;display: inline-block;}
a:hover, a:active {background-color: red;}

Bookmarks can be useful if a web page is very long. To create a bookmark, first create the bookmark, then add a link to it.
When the link is clicked, the page will scroll down or up to the location with the bookmark.
-Use the id attribute (id="value") to define bookmarks in a page. Usually in headings.
-Use the href attribute (href="#value") to link to the bookmark

HTML IMAGES

You can use the style attribute to specify the width and height of an image. For example:
'img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;"'or
'img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600"'
However, style attribute are more suggested. It prevents styles sheets(CSS) from changing the size of images.

To use an image as a link, put the "img" element inside the "a" element.

Use the HTML "img" element to define an image
Use the HTML src attribute to define the URL of the image
Use the HTML alt attribute to define an alternate text for an image, if it cannot be displayed
Use the HTML width and height attributes or the CSS width and height properties to define the size of the image
Use the CSS float property to let the image float to the left or to the right

The HTML "map" tag defines an image map. An image map is an image with clickable areas.
The areas are defined with one or more "area" tags. Here's an example:
'img src="workplace.jpg" alt="Workplace" usemap="#workmap"' <--to find image
'map name="workmap"' <--to display map image
'area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm"'<--x,y value starts from left top corner
'area shape="circle" coords="337,300,44" alt="Coffee" href="coffee.htm"'<--44 is the radius of thr circle from the center
'area shape="poly" coords="140,121,181,116,204,160,204,222,191,270,140,329,85,355,58,352,37,322" href="croissant.htm">
'/map'

A clickable area can also trigger a JavaScript function. For example:
'map name="workmap"'
'area shape="circle" coords="337,300,44" href="coffee.htm" onclick="myFunction()"' 'script'
function myFunction() {
alert("You clicked the coffee cup!");
}
'/script'

A background image can be specified for almost any HTML element.
Depending on which element you want to put the background, add 'background-image' in style. For Example:
'style'

background-attachment: fixed;<--to make sure the image will keep its original proportions
background-size: 100% 100%;<--to stretch the image to fit the entire element.Use"cover"if don't want to stretch
}
