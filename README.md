### Setting up locally

1. Clone the project and move to folder.

  ```shell
  git clone https://github.com/NutNut17/thulite-project.git
  cd thulite-project
  ```

2. Install dependencies.

  ```shell
  npm install
  ```

3. Run dev server.

  ```shell
  hugo server
  ```

4. Finally, visit `http://localhost:1313/` from your browser.
